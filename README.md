# Avalon Multiplayer

Using Colyseus Game Server (v0.12.x)

# Running the server locally

You'll need two terminal windows open to be able to run this. One for the
client, and one for the server:

**Running the server**

```
cd server
npm install
npm start
```

**Running the client**

```
cd client
npm install
npm start
```
